# 📊 Data Analytics Practice using Python

This repository contains my hands-on data analytics practice using Python and Pandas. The notebook focuses on data preprocessing, data cleaning, exploratory data analysis (EDA), and statistical operations on an E-commerce Orders dataset.

## 📌 Project Overview

The notebook demonstrates the complete workflow of analyzing a dataset, including:

- Loading datasets using Pandas
- Exploring dataset structure
- Handling missing values
- Data cleaning
- Descriptive statistics
- Data manipulation
- Feature analysis
- Basic exploratory data analysis (EDA)

## 🛠️ Technologies Used

- Python
- Pandas
- NumPy
- Jupyter Notebook

## 📂 Dataset

**Dataset Used:**
- Ecommerce Orders Dataset

## 📚 Topics Covered

- Importing libraries
- Reading CSV files
- Exploring datasets (`head`, `tail`, `shape`)
- Detecting missing values
- Handling null values
- Mean and Median imputation
- Dropping missing values
- Statistical summary
- Data inspection

1. Clone this repository.
2. Install the required libraries:

```bash

```

3. Open the notebook:

```bash

```

4. Run all cells sequentially.
